Artifact Examples Overview

This page points to a subset of artifacts from an example project developed in accordance with the Rational Unified Process, guidelines and templates.

Example artifacts are provided to show their evolution from phase to phase during the execution of the project.

The first set of example artifacts is based on a fictional project to develop an on-line course registration system for Wylie College. The project is intended to replace the existing course registration system with a state-of-the-art on-line system that allows student and professor access through PC clients.
